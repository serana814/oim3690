- If you inspect a website and see display: flex on a container with 5 items. The items are stacked vertically intead of in a row. This is caused by the property called flex-direction: column — it changes the flex main axis to vertical so the items stack top-to-bottom (also possible: flex-direction: column-reverse or the shorthand flex-flow).
