#### To-do app learning log
- The <script> tag is at the bottom of the HTML file, just before </body>.
- getElementById() is used in three places:
    - 'todo-form': selects the form for adding todos
    - 'todo-input': selects the input field for new todos
    - 'todo-list': selects the <ul> where todos are displayed
- There is no document.querySelector() 
- addEventListener() is used in 2 places:
    - On the form, it listens for the 'submit' event to add a new todo
    - On each todo's checkbox, it listens for the 'change' event to toggle the completed state.
- How the code changes the DOM: 
    - Creates elements with document.createElement (li, div, input, span, button)
    - Adds elements using appendChild
    - Removes elements with removeChild
